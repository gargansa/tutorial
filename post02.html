Making bash work for you with rc files

Alias prompt and functions

Posted by Robert Crum last updated October 20, 2019

What is the bashrc and zshrc file

These files are loaded when you start a terminal session.

Used to personalize your prompt, define alias to shorten commands, and store commonly used functions to make life easier.

Two types of terminal sessions are

bash The default on macOS 10.3 - 10.14 (including El Capitan 10.11, Sierra 10.12, High Sierra 10.13, and Mojave 10.14)

zsh which is default on macOS 10.15 Catalina

Bash

Where are bashrc files?

cd /etc

How to edit bashrc files in the terminal

sudo vim /etc/bashrc

Set bash as default in terminal.

chsh -s /bin/bash && exit

Letting VS Code know to use bash as default.

Open a terminal by holding control + ~

Click on the shell button

Select default Shell

Choose the shell you want from the list

Close the terminal control + b and reopen with control + ~

Adding color to bash

Color options available for bash with 256 colors enabled.

Creating a colorful prompt

Edit the bashrc file in vim using the instructions above

Copy this code

NC='\033[0m'
FG=37
BG=53
COLOR= '\033[38;5;${FG};48;5;${BG}m'
PS1=\[$COLOR\]\D{"%m-%d-%y"}\w\[$NC\]

Paste it into the vim editor in the terminal

Changing oh_my_zsh Themes

Visit Oh My Zsh on github to find a theme you like

Open the terminal and type sudo vim ~/.zshrc

Use the arrow keys to move down to the line that says ZSH_THEME

i key

Delete the theme within the " "

Type in the theme you want.

esc key

Type :wq! and hit Enter

If you make a mistake when editing this file type :q!

Creating Alias

Open the bashrc or zshrc file using the edit instructions above

Bash

alias rc='sudo vim /etc/bashrc && source /etc/bashrc'

To save this ESC then :wq!

Reload the saved file with source /etc/bashrc

Now anytime you type rc it will open your bash file into vim editor and automatically reload the file when you exit

Zsh

alias rc='sudo vim ~/.zshrc && source ~/.zshrc'

To save this ESC then :wq!

Reload the saved file with source ~/.zshrc

Now anytime you type rc it will open your bash file into vim editor and automatically reload the file when you exit

Creating Functions

#Function that takes an input when called

#Executed by typing the function name followed by value

#Example: cake 2

cake(){

for X in $(seq 1 $1);

do

echo "Eats " $X " piece of cake"

done

}

#Function that requests an input

#Executed by typing just the function name it will then prompt for input.

#Example: who

who(){

echo "Please enter your name:"

read RESPONSE

echo "Hello" $RESPONSE

}

#Function to display available colors

#Executed by typing colors in terminal

colors(){

for EACH in {0..255};

do

printf "\x1b[48;5;%sm%3d\e[0m " "$EACH" "$EACH"

if (( $EACH % 15 == 0 )); then

printf "\n";

fi

done

for EACH in {0..255};

do

printf "\033[38;5;%sm%3d\e[0m " "$EACH" "$EACH"

if (( $EACH % 15 == 0 )); then

printf "\n";

fi

done

}
