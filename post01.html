Intro to the bash language

A short list of useful bash commands

Posted by Robert Crum last updated November 4, 2019

What is Bash

A Unix shell and command language that lives inside terminal.

    Manual

  • man When used before a command displays info about the command
  • man ls Displays the manual for the ls command.

    • Navigating through folders

  • pwd Displays what folder you are curently in.
  • cd .. Moves back one directory
  • cd ~ Moves to your home directory
  • cd / Moves to root directory
  • cd - Return to previous directory you were in
  • cd ../otherFolder Move back one directory and then into a different folder
  • Tab key - Use to autocomplete file and directory names.
  • Tab Tab Double tab for list of files and folders that match.
  • ls List contents of directory
  • ls -larth Display more info about files and folders
  • ls -ld Display more info about directory you are currently in.

    • File and Folder Creation/Deletion

  • touch filename.txt Creates a file
  • mkdir folder Create a Folder
  • rm filename.txt Remove a file
  • rm -r folderName Remove a folder named folder along with everything inside
  • rm * Removes all files within the current directory - Be careful with this one.

    • Move, Copy, and rename

  • mv filename.txt /newLocation/ moves filename.txt to /newLocation
  • mv * ../differentFolder move every file in this folder into a sibling folder
  • cp filename.txt /folder/ Copies the file to a new location
  • rename old new oldname.txt Renames oldname.txt to newname.txt
  • rename F f Facebook* Change all F in files that start with Facebook to f

    • File viewing

  • less filename.txt Opens filename.txt into the less program
  • commands within less
  • q exit
  • arrow keys move through file
  • G Jumps to the bottom of the file
  • g Jumps to the top of the file
  • ? Search up from current location in file
  • / Search down from current location in file
  • N Go to the previous search result location
  • n Go to the next search result location

    • File editing

  • vim filename.txt Opens filename.txt into the vim program
  • use arrow keys move through file

  • When not in insert mode
  • To Quit without saving :q! followed by Enter
  • To Save and Quit :wq! followed by Enter
  • dd Delete line - Similar to Cut
  • 3dd Delete # of lines starting with current line
  • yy Copy line
  • 3yy Copy # of lines starting with current line
  • p Paste the last copied or deleted line/lines
  • i Switch to insert mode

  • When in insert mode
  • Esc Return from insert mode

    • Sudo

  • whoami Sometimes you may need to run commands as other users such as root.
  • sudo su root Become the root super user

    • Permissions

  • chmod 777 folder Fully opens the permissions of a folder
  • chmod 777 filename.txt Fully opens the permissions of a file
  • sudo chmod 751 filename.txt Change the permissions for files you dont own
  • ls -larth displays these permissions

  • Examples of how the permissions display with ls -larth
  • -rwxrwxrwx
  • drwxrwxrwx
  • -rw-r-xr--
  • --wx-w---x
  • Value of each location
  • rwx

    • Ownership

  • chown username:group folder Change the ownership of a folder
  • chown username:group filename.txt Change the ownership of a file
  • sudo chown username:group filename.txt Change the ownership of a file when the file does not belong to your user
