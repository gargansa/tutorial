Intro to the bash language

A short guide to bash commands

Posted by Robert Crum on October 6, 2019

What is Bash

A Unix shell and command language that lives inside terminal.

    General Useful

  • man Manuel - use before commands
  • man ls Displays the manual for the ls command.
  • * Wildcard

    • Navigating through folders

  • pwd Print Working Directory
  • cd Change Directory
  • cd .. Move to parent directory
  • cd ~ Move to home directory
  • cd / Move to root directory
  • cd - Return to previous directory
  • cd ../otherFolder Move back one directory and then into otherFolder
  • Tab key Use to autocomplete file and directory names. double click for list of options.
  • ls List contents of directory
  • ls -larth More info l=long format; a=all files display hidden; r=reverse order oldest first; t=sort by time; h=byte suffixes

    • File and Folder Creation/Deletion

  • touch Create a file
  • touch filename.txt Creates a file with the name filename.txt
  • mkdir Make Directory
  • mkdir folder Makes a folder with the name folder
  • rm Remove
  • rm filename.txt Remove the file named filename.txt
  • rm -r folder Remove the folder named folder along with everything inside
  • rm * Removes all files within the current directory - Be careful with this one.

    • Move and rename

  • mv Move a file or folder
  • mv filename.txt /location/newFolder/ moves filename.txt to /location/newfolder
  • mv * ../differentFolder move every file in this folder into a sibling folder named differentFolder
  • rename modify the name of files
  • rename old new oldname.txt Rename oldname.txt to newname.txt
  • rename F f Facebook* Change all F in files that start with Facebook to f

    • File viewing

  • less Opens a file in read-only mode
  • less filename.txt Opens the file named filename.txt into the less program
  • commands within less
  • q exit
  • G go to bottom of file
  • g go to top of file
  • ? Search up in file
  • / Search down in file
  • n next search
  • shift n previous search

    • File editing

  • vim Opens a file in edit mode
  • vim filename.txt Opens the file named filename.txt into the vim program
  • commands within vim
  • :q! exit without saving (colon quit bang)
  • :wq! exit with saving (colon write quit bang)
  • arrow keys move through file
  • i Switch to insert mode
  • dd Delete line when not in insert mode

    • Sudo (Please do what i asked)

  • whoami Displays the user you currently are.
  • sudo su root Become the root super user

    • Permissions

  • chmod Change mode
  • chmod 777 folder Fully opens the permissions of a folder
  • chmod 777 filename.txt Fully opens the permissions of a file
  • sudo chmod 777 filename.txt Change the permissions when you don't have permission
  • You can see these permissions by typing ls -larth in a folder
  • -rwxrwxrwx (- = file)
  • drwxrwxrwx (d = directory)
  • chmod 764 filename.txt sets the permmissions of filename.txt to
    • 7(rwx)Read/Write/Execute permissions for the user the file belongs to
    • 6(rw-)Read/Write permissions for group the user is a part of
    • 4(r--)Read permissions for everyone

    • Ownership

  • chown Change ownership
  • chown username:group folder Change the ownership of a folder to username:group
  • chown username: folder Leaving out the group will choose the default for that user
  • chown username:group filename.txt Change the ownership of a file
  • sudo chown username:group filename.txt Change the ownership of a file when you don't have permissions
